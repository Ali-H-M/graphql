# GraphQL Profile Dashboard

A web application that displays user profiles, XP progress, audit ratios, and project statistics from the GraphQL API.

## Tech

- **HTML5** - Structure
- **CSS3** - Styling with gradients, backdrop filters, and responsive design
- **Vanilla JavaScript** - No frameworks, pure ES6
- **SVG** - Interactive charts and graphs
- **GraphQL API** - Data from learn.reboot01.com
- **LocalStorage** - JWT token persistence for sessions
